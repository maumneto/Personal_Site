Linhas de Pesquisa

Internet das Coisas (Internet of Things - IoT)

Os avanços das tecnologias de comunicação sem fio e de sistemas embarcados possibilitaram o surgimento do paradigma de Internet das Coisas. A Internet das Coisas (Do inglês Internet of Things, IoT) pode ser definida como um conjunto de dispositivos com poder computacional para captar dados do ambiente, processa-los e envia-los para outros dispositivos, fornecendo serviços aos seus usuários. Esses dispositivos são comumente chamados de objetos inteligentes pois, na maioria das vezes, são objetos do dia-a-dia que ganham poder de processamento e de comunicação. O objetivo desta linha de pesquisa é estudar e criar aplicações voltadas a este novo paradigma que está cada vez mais emergente. Como interesse maior são estudadas as aplicações de e-health e aplicações do tipo smart-*.

Redes de Sensores Sem Fio (Wireless Sensor Network)

As Redes de Sensores sem Fio (RSSF) podem ser definidas como um conjunto de pequenos dispositivos (comumente conhecidos como nodos) que possuem poder de comunicação e de sensoriamento para monitorar um determinado ambiente. Esta rede tende a ser autônoma e possui um alto grau de cooperação entre os demais nodos para exercer a tarefa definida. O objetivo desta linha de pesquisa é o estudo de novos protocolos para as RSSF, como também, o estudo de meios para resolução de problemas que ocorrem nas RSSF (ex: autonomia da rede, economia de bateria dos nodos, protocolos de comunicação mais eficientes, etc).



Ciência dos Dados (Data Science)

Atualmente tem-se uma grande disponibilidade de dados que podem ser facilmente acessíveis por diversas pessoas, pesquisadores e empresas. Entretanto, alguns dados isolados podem não gerar uma informação útil aos usuários finais. A ciência dos dados (Do inglês, Data Science) tem como objetivo principal a análise e o estudo dos dados captados, sejam estruturados ou não. Com base nestas análises, é possível tomar decisões ou informações que antes não eram possíveis com os dados brutos. É uma área interdisciplinar que abrange a estatística, economia, engenharia, dentre outros. O objetivo desta linha de pesquisa é gerar análises de dados para diversas áreas de Internet das Coisas (Do inglês, Internet of Things, IoT), fazendo com que as aplicações de IoT possam tomar decisões corretas. É especificamente estudado os dados de aplicações de e-Health e de fluxo de dados de uma rede IoT.



E-health

Umas das vertentes da Internet das Coisas que está em constante crescimento atualmente é a E-health. As aplicações de E-health utilizam a internet em conjunto com outras tecnologias da informação com o objetivo de melhorar o tratamento de uma paciente e as condições dos processos clínicos, diminuindo o custo do sistema de saúde. Existem diversas dimensões de aplicações dentro do conceito de e-health. O objetivo desta linha de pesquisa é a resolução de problemas de monitoramento de pacientes que estão em grupos de riscos (ex.: cardíacos) por meio do uso de dispositivos vestíveis (do inglês, wearables).
